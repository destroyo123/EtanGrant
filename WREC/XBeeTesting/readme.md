# Instructions
## Pre-Requisites:
- VSCode with the Platformio IDE extension and its dependencies installed in VSCode.
- Python 3
  - pip install pyserial
  - pip install websockets
 ## Instructions
 - Download the entire parent folder (parent folder contains the 'platformio.ini' file. Example: download "XBee-XNO55-Test_Uno")
 - Using the "platformio IDE" extension, click "open" -> "open project" -> locate that same parent folder "Open"
 - wait for PlatformIO to download everything and set up. It will take a few minutes. (progress/activity shown in the bottom right corner of VSCode.
 - Ready to go!
